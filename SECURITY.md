# Security Policy

## Reporting Security Vulnerabilities

The security of this project is taken seriously. If you believe you have found a vulnerability in the Albion Data Client, please report it responsibly and do not disclose it publicly until it has been addressed.

## Reporting Process

To report a security vulnerability:

1. Do NOT open a public GitHub issue
2. Email the project maintainers with details about the vulnerability
3. Include steps to reproduce if possible
4. Allow reasonable time for the maintainers to respond and prepare a fix

## Supported Versions

Security updates are provided for the latest stable release. Users are encouraged to update to the latest version promptly.

## Security Considerations

When using the Albion Data Client:

- Ensure you are running an official release or building from verified source code
- Keep your system and dependencies up to date
- Follow the Terms of Service for Albion Online
- Monitor for security advisories

## Acknowledgments

We appreciate responsible disclosure and efforts to improve the security of this project.
